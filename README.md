# Holoboard
Hololens app - Interactive Blackboard

# Getting Started
- Clone repository (needs Unity 2017.2)
- Import Holotoolkit (https://github.com/Microsoft/MixedRealityToolkit-Unity/releases/tag/v1.2017.2.0) in Assets folder
- Have fun !
